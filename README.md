# Web Scrapping - Cotação do Dólar

Este repositório contém um script em Python para realizar web scrapping de cotações do dólar de várias fontes e inserir esses dados em um banco de dados PostgreSQL.

## Conteúdo

- `CotacaoDolar.ipynb`: Notebook Jupyter contendo o código para importar, tratar e inserir dados de cotações do dólar no banco de dados PostgreSQL.

## Dependências

Para executar este notebook, você precisará das seguintes bibliotecas:

- requests
- beautifulsoup4
- pandas
- psycopg2

Você pode instalar essas dependências usando `pip`:

```sh
pip install requests beautifulsoup4 pandas psycopg2
